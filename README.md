# MHXY PK Vision Assistant

梦幻西游手游 PK 合规视觉辅助/训练助手研究项目。

## 重要边界

本项目不开发、也不接受以下能力：

- 读取、篡改或注入游戏客户端内存/网络封包/进程；
- 绕过反外挂、自动化代打、自动决策或自动操作；
- 显示游戏原本不向玩家公开的对方真实血量、蓝量、技能冷却、状态等隐藏信息；
- 任何违反《梦幻西游》手游、网易游戏用户协议或赛事规则的插件。

用户最初想做“PK 时看到对方血条”的插件。该目标如果意味着获取游戏未公开的真实数值，属于高风险作弊/外挂方向，本仓库将其重构为合规方向：

1. 赛后复盘：导入录像/截图，人工标注或 OCR 已显示信息，估算战斗节奏与伤害区间。
2. 训练沙盒：用手动输入或公开可见事件构建血量区间估计器，帮助理解集火、守尸、回复窗口。
3. 观战/直播辅助：只在获得授权的录像、直播画面或自有素材上做非侵入式屏幕分析。
4. 无游戏交互：工具不向游戏客户端发送输入，不读取进程，不连接游戏网络。

## MVP 方向

第一版建议做“PK 录像复盘 + 血量区间估算器”：

- 输入：本地视频/截图、人工录入基础属性、回合事件表。
- 输出：每个单位的血量区间、伤害/治疗事件时间线、关键回合复盘摘要。
- 技术：Python、OpenCV/Pillow、可选 OCR、Streamlit/FastAPI 前端。
